Projeto - Analise dos dados do IMDB

Glossário deste documento:
    Data Warehouse - DW;
    Extract, transform, load - ETL;
    Online Analytical Processing Cubes - OLAP;

Étapas do projeto: 
    -   1ª Etapa: Solicitação de permissão para analisar os dados do IMDB.
    -   2ª Etapa: Obtenção da base de dados do IMDB, caso permissão sejá concedida.
    -   3ª Etapa: Analise da estrutuda dos dados do IMDB. 
    -   4ª Etapa: Criação de dicionário de dados, caso necessário.
    -   5ª Etapa: Construção do ETL.
    -   6ª Etapa: Exportar dados para o DW.
    -   7ª Etapa: Construção do OLAP.
    -   8ª Etapa: Construção dos dashboards e paineis.

Cronograma do projeto:
    -   1ª Etapa: O projeto se dá início após obtenção da permissão e dos dados (1 à 2 Semanas);
    -   2ª Etapa: O projeto se dá início após obtenção da permissão e dos dados (1 à 2 Semanas em conjunto com a 1ª Etapa);
    -   3ª Etapa: Analisar as estruturas dos dados, suas ligações e como poderia se estruturar a staging_area, além de sua criação (1 Semana);
    -   4ª Etapa: Criar o dicionário de dados, mediante o que foi analisado na 3ª Etapa (1 dia);
    -   5ª Etapa: Construção do ETL, já consumindo os dados da staging_area (2 Semanas);
    -   6ª Etapa: Exportando os dados do ETL para o DW (2 Dias);
    -   7ª Etapa: Construindo o OLAP (2 Dias);
    -   8ª Etapa: Construindo os dashboards e paineis (1 Semana);
